# aws-flask-rds-project
Simple web application deployed on AWS using:

- EC2
- RDS MySQL
- Flask
- Security Groups

## Features
- Add users
- Store data in MySQL
- Read data from database

## AWS Services Used
- EC2
- RDS
- IAM
- Security Groups

## Architecture

User -> EC2 Flask App -> RDS MySQL

## Run Locally

```bash
pip install -r requirements.txt
python3 app.py
```
